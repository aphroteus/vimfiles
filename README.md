# vimfiles
My vimrc

## Installing on Windows

    cd /d %UserProfile%
    git clone https://github.com/aphroteus/vimfiles.git
Open vim or gVim and run `:PluginInstall`

## Installing on Linux

    cd ~/
    git clone https://github.com/aphroteus/vimfiles.git
    mv ./vimfiles ./.vim
Open vim and run `:PluginInstall`
